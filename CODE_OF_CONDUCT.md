# Code of Conduct

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be respectful, constructive, and patient with other contributors.
- Assume good faith.
- No harassment, personal attacks, or discriminatory language.
- Critique the code, not the person.
- If you see behavior that violates these norms, report it privately to
  the project maintainer via the channel listed in `SECURITY.md`.

The maintainer reserves the right to remove comments, close issues or
pull requests, and ban contributors who do not adhere to this code.
